# quantaxisdesktop

> quantaxis desktop
#### Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron application for production
npm run build

# build web 
npm run build:web

```

---

打包前强烈推荐先安装阿里的淘宝npm
```
npm install cnpm -g
```

使用cnpm 全局安装electron 速度非常快
```
cnpm install electron -g

```

再使用cnpm安装依赖项

```
cnpm install 
```
